Command list
  • !help - Prints the command list.
  • !hi - Writes a random greeting to author.
  • !greet - Writes custom greeting attributed to author.
  • !setgreet <str> - Saves custom greeting for author.
  • !group <server> - Sets up a private group on the given server.
  • !pubgroup <server> - Sets up a public group on the given server.
  • !find <username> - Searches for the user in current server.
  • !stream - Gives the current stream link.
